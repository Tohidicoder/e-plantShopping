# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React-based online plant shop application.

This project allows users to:
- Browse different types of house plants
- View plant details including images, names, and prices
- Add plants to a shopping cart
- Increase or decrease plant quantities
- Remove items from the cart
- View the total cart cost

## Technologies Used

- React
- React Hooks
- Redux Toolkit
- JavaScript
- CSS

## Project Overview

Paradise Nursery provides a simple and user-friendly shopping experience for customers who love indoor plants.
